## Application

Language and frameworks in backend: python, fastApi + pydantic + sqlAlchemy

## Running app

Running application:

```shell
poetry install

docker run -d -p 5433:5432 --name postgres -e POSTGRES_HOST_AUTH_METHOD=trust postgres
poetry run uvicorn app.main:app --port 8080

```

# Testing (swagger):
